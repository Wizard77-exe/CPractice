# Inventory System
---
A CLI-based application that lets user add Items and quantities, save the items and quantity to a file
for persistence.
---
### Features
1. Add Items and Quantity for each Item
2. Display all Records
3. Update Records
4. Delete Records
### Required Modules
1. <stdio.h>
2. <string.h>
3. <stdlib.h>
---
### Cloning
1. Clone the Repository
`git clone https://github.com/Wizard77-exe/CPractice`
2. Go to the Directory of the cloned Repo
`cd CPractice/BasicProjects/inventory-system`
3. Compile the C code
`gcc inventory-system.c -o inventory-system`
4. Run the executable
`./inventory-system`
