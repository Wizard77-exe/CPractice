# To-Do List
It's a CLI-based todo list with the basic functions. No file handling yet to save the list on a file to make it recoverable.
Just some sort of to-do list that doesn't save your tasks everytime you exit.
---
## Features
- Adding tasks
- displaying tasks
- deleting tasks
- marking a tasks as done
- displaying finished tasks
- reset finished Tasks (deleting all the finished tasks)
---
## Required Modules
- <stdio.h>
- <stdlib.h>
- <string.h>
---
### Cloning
```bash
git clone https://github.com/Wizard77-exe/CPractice.git
cd CPractice/BasicProjects/todolist
gcc todolist.c -o todolist
./todolist
```
