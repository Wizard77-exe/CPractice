# HANGMAN GAME
Hangman game is a popular game where a player guesses a letter, and if a letter is in the word, the letter will show up on its 
position in the word
```bash
# let's say the word is CLOUD
_ _ _ _ _ 

# user guesses 'O'
_ _ O _ _

# user guesses 'D'

_ _ O _ D

# and that goes on.
```
